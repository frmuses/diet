Philosophy & Goals

Diet provides all the necessary high-tech tools & mechanisms to build the next google, facebook, wikipedia, airbnb or anything. There is a long road ahead of diet but the aim is to make the fastest and most flexible javascript server ever.

What's the difference?

  • It's node.js, and asynchronous
  • Efficient and blazing fast
  • Fully Object Oriented
  • Extremely fast development - build apps in hours.
  • Includes 90% of the most common tools that apps need. - data store, email, accounts, notifications, tasks, routing etc...
  • Simplicity and Human readability is key in default APIs & Functions.
