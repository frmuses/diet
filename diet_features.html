Features

Documentation is coming soon but feel free to start early hacking with diet.

What you can do with Diet?

  • MySQL as Data Store - No-SQL like basic Create, Read, Update, Delete Functions
  • Simple yet advanced Page Routing - regex support
  • Built in Email Support - Over SMTP (Mandrill, Sendgrid, Gmail etc..)
  • Built in HTML Parser - (ECT based)
  • Built in Real Time Notifications - (COMET)
  • Built in Cookies - diet cookies, yummy!
  • Built in Asynchronous handler - nextjs
  • Built in Task/Schedule Module
  • Built in Accounts API
  • Built in File Uploading API
  • Built in Translation Engine with Dictionary table in MySQL
  • Multi Subdomain/domain Support - multiple projects on the same IP address
  • Real-time File Uploader Module - (500mb/sec upload, based on formidable)

Things you can build with Diet

You can build any web & mobile app with diet but since node.js is event driven you can build real-time apps, websites in a breeze like:

  • Social networks
  • CMS/Blog engines
  • Chat clients
  • REST API's
  • File Servers
  • Real-time games
  • Real-time analytics/stats
  • Real-time feeds
  • Real-time stock tickers
